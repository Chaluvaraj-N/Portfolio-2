# Chaluva&#39;s Portfolio

## Overview
Modern, responsive single-page portfolio website showcasing Full-Stack Development &amp; AI/ML skills.

**Tech Stack**: Pure HTML5, CSS3, Vanilla JavaScript

## Features
- Dark mode toggle (CSS-only)
- Smooth animations &amp; particles
- Fully responsive (mobile-first)
- Sections: Hero, About, Skills, Projects, GitHub, Experience, Contact

## Files
- `Portfolio.html` - Main portfolio

## Quick Start
```bash
# Windows - Open portfolio
start Portfolio.html

# Deploy to GitHub Pages/Netlify (drag &amp; drop)
```

## Customize
1. Edit hero title/subtitle in `Portfolio.html`
2. Update projects/experience sections
3. Modify skills chips
4. Change contact links/email
